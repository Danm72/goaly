# Meeting Notes Template

## Directory Structure

Each call should have its own file with the naming format: `YYYY-MM-DD-call-topic.md`

## Template

```
# Call Notes: [Date] - [Topic]

## Participants

- [Name] ([Role])
- [Name] ([Role])

## Agenda

- [Item 1]
- [Item 2]
- [Item 3]

## Discussion Points

- [Key Point 1]
  - [Details]
- [Key Point 2]
  - [Details]

## Decisions Made

- [Decision 1]
- [Decision 2]

## Action Items

- [ ] [Action Item 1] - [Owner] - [Due Date]
- [ ] [Action Item 2] - [Owner] - [Due Date]

## Next Steps

- [Next Step 1]
- [Next Step 2]

## Additional Notes

[Any other relevant information, observations, or follow-up questions]
```
