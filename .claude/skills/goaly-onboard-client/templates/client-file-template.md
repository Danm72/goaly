# Client File Template

Write to `notion-mirror/clients/<client-slug>.md`:

```yaml
---
title: "Company Name"
status: Lead
engagement_type: "Fractional CTO"
rate: 150
area:
  - [industry tags]
---

## Context

Company details, tech stack, platform info.
Hours/month: [TBD or specific]
Start date: [TBD or specific]
Engagement terms: [retainer structure, billing, etc.]

### Communication Cadence

- Meeting frequency: [weekly/biweekly/monthly/ad-hoc]
- Preferred channel: [email/WhatsApp/Slack]
- Status updates: [frequency and format]
- Time tracking: [sheet URL]

## Strategy Notes

Red flags observed during screening.
Engagement posture: [Expansive/Cautious/Protective]
What NOT to offer: [boundaries]
Pricing strategy: [any special terms]
```

## Fields to Collect (via AskUserQuestion)

- Company name
- Engagement type (Fractional CTO / Retainer / Project / Advisory / Pilot)
- Rate (default [Your Rate]/hr)
- Industry/tech stack tags
- Key context (what they do, how they found [Owner])
